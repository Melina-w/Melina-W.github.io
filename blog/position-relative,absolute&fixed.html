Relative, absolute, and fixed positioning

Positioning sets how an element is positioned in a document, that is to say make elements behave differently. For example, sitting one element on top of another one or at the very top of the page and to do this, the top, right, bottom and left properties will be necessary.

The position property can take different values like relative, absolute and fixed but you might be wondering what are the different between them. Let’s get started!

{position: relative;}

An element with position relative will follow the normal flow of the webpage, so the element is relative to itself. For instance, let’s say we want to set position relative to another element on the page.

As shown on the img, we need to set a positioning attribute (top, left, bottom, right) since otherwise, it won’t have any effect on the positioning at all. In this case the attribute top:50px was set which means the element will shift its position 50px down from where it would normally be.

{position: absolute;}

This is a powerful type of positioning as the element with absolute position is taken out of the normal flow of the website and other elements are position as if the absolute element does not exist. Another characteristic to have in mind is that the absolute positioned element is positioned relative to its nearest positioned ancestor (i.e, the nearest ancestor that is not static)

A we can see on the previous image, the child element although being still inside the parent, it is positioned at the top-left of it. This is because we set the top and left attribute to 0px on the child attribute and position relative on the parent element. If we remove the position of the parent, the child will be moved to over the parent, following the grand-parent element (the body) which is as far as it can go.

{position: fixed;}

A fixed position element will work exactly the same way as absolute position with the difference that the element in this case is position relative to the viewport, or browser window itself. Fixed positioning is commonly used to make navigation bars that remain visible at all times regardless of the pages scroll position.
